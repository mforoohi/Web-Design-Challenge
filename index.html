Project Summary





The purpose of this project was to analyze how weather changes as you get closer to equator. to accomplish this analyses, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

After Assembling the dataset,we used Matplotlib to plot variuos aspect of the weather vs. latitude. actors we looked at included: tempurture,cloudiness,wind speed, and humidity.This site provides the source data and visualization created as part of the analysis, as well as examplanations and descriptions of any trends and correlations witnessed.
